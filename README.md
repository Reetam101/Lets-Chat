# Lets-Chat
## [Live version](https://lets-chaat.herokuapp.com/)
### A chat app built using MERN stack.

## Features
- One on one chat
- Group chat
- add/remove members to/from group

## Tech Stack Used -
- React.js and Chakra UI
- Express
- MongoDB
